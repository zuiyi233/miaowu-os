# 恐惧升级

恐怖升级要像螺丝收紧，不像音量放大。最稳的路径是让角色一步步失去安全解释，再失去安全选择。

## 常用升级阶梯

1. 轻微不对劲
2. 发现这不是偶然，而是模式
3. 原本可用的安全解释失效
4. 威胁变得具体，角色必须应对
5. 场景结束后仍有残留后果

## 写作动作

- 每次升级都让角色少掉一个安慰自己的理由
- 让异常从感官层面进入行动层面，再进入结果层面
- 如果要慢烧，就让“证明它真的存在”的门槛越来越低
- 升级之后要有余波，例如伤口、污染、关系破裂、规则确认

## 判断标准

- 后一层比前一层不仅更响，而且更贵
- 角色的应对方式在升级中不断失效
- 读者会感觉到世界本身变得更不可信

## 常见误区

- 每一层都只是更大声、更血腥
- 角色一直重复同一种害怕反应
- 有冲击，没有残留
