# 揭晓顺序

推理的揭晓往往不是“宣布答案”，而是逐层拆除旧解释。顺序安排得好，真相会像结构性坍塌一样出现，而不是像老师公布标准答案。

## 常用顺序

1. 先推翻最顺手的错误解释。
2. 再显出此前被忽略的模式。
3. 把动机、手法、时机和机会连接起来。
4. 最后展示真相带来的情感、关系或制度代价。

## 写作动作

- 不要等到最后一分钟才给读者理解真相所需的关键拼图
- 每次小揭晓都应改变怀疑方向、人物站位或行动策略
- 如果答案很晚出现，前面的线索就必须更能承重
- 真相出来后不要立刻收尾，留出“知道之后怎么办”的压力

## 判断标准

- 揭晓之后，读者会想回翻前文，而不是抱怨信息不足
- 每一层揭晓都把局面推向更难处理的位置
- 真相不仅解释了事件，也重写了人物关系或道德评价

## 常见误区

- 一口气说明全部答案，缺少层次
- 还没给足证据就让角色突然顿悟
- 只揭晓“谁做的”，却没让“为什么”和“代价”落地
