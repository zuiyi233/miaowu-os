# 悬念核心

悬念要同时满足两件事：有活着的问题，也有拖延的代价。少一个，读者都会松掉。

## 五个核心杠杆

- 接下来可能发生什么
- 谁正在暴露，暴露到什么程度
- 角色还不知道什么
- 读者部分知道什么，或被引向什么判断
- 距离不可逆损失还有多久

## 常用推进方式

- 让不确定性更具体，而不是更模糊
- 让选择更少，而不是让信息原地打转
- 让代价更高，例如时间更少、见证者更多、误判后果更重
- 暴露一个新的危险或新的误读路径
- 给角色一个暂时看似可行、其实更糟的选择
- 给主角再加一层必须同时处理的麻烦，让局面更像多盘并转而不是单点冲刺
- 让对手或外部世界反应，而不是只有主角单边行动

## 写作动作

- 进入场景前先写下“本场的悬念句”
- 中段安排一次压力变化，让角色不得不修正判断
- 后段避免纯解释，尽量用行动、被打断、失败尝试来制造张力
- 如需延迟答案，先交代为什么现在拿不到，而不是硬拖

## 判断标准

- 读者能说出自己在等什么
- 角色不是被动挨等，而是在带着不完整信息行动
- 场景结束后，局面更窄、更险或更急
