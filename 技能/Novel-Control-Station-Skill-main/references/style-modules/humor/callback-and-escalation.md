# 回调与升级

回调不是重复一遍笑点，而是让旧材料在新压力下返回。升级也不是把场面写得更吵，而是让代价越来越难收拾。

## 强回调模式

1. 先埋一个小而清晰的滑点。
2. 在更高压力场合让它再次出现。
3. 让第三次出现不只是好笑，还要揭示人物、关系或局势真相。

## 常见升级阶梯

1. 私下尴尬
2. 被旁人看见
3. 地位受损或计划受阻
4. 情感后果或情节后果落下

## 写作动作

- 每次回返都改变至少一项条件：场合、观众、 stakes、情绪或关系位置
- 回调前先让读者记得，不必大声提醒，只要在场景里自然带过
- 升级时优先增加暴露风险和补救成本
- 如果笑点已完成使命，就收，不要贪第四次第五次

## 判断标准

- 第二次出现比第一次更难堪或更危险
- 第三次出现带来的是结果，不只是重复识别
- 即使读者忘记术语“回调”，也会感觉这件事终于撞回来了

## 常见误区

- 只重复台词，不升级压力
- 用旁人提醒代替自然回返
- 一味加大音量、动作、人数，却没有增加成本
- 明明已经落地，却舍不得收手
