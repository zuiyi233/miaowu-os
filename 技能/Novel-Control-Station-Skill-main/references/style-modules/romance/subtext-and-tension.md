# 潜台词与张力

很多有效的爱情场景里，人物嘴上在谈别的，真正交换的是试探、照顾、占有、退让和脆弱度。

## 常用动作

- 绕着感觉说话，不直接命名感觉
- 让照顾伪装成不耐烦、控制欲或规矩
- 让一个具体动作承担情绪，例如递水、挡人、送回家、删消息、停顿
- 可以让人物只开口要一个更小的东西，实际想得到的是更大的确认
- 让双方暴露程度不对称，从而制造张力

## 写作动作

- 先写清 literal topic，再写清 real topic
- 若该场景依赖潜台词推进，可以安排一个人物差点说穿又收回的瞬间
- 用身体反应、动作选择和注意力偏移补足未说出口的内容
- 如果张力要升，不一定靠拥抱或亲吻，也可以靠更危险的诚实
- 如果场景已经涉及明确边界、拒绝或同意，潜台词不能拿来抹糊这些事实

## 检查本章是否改变了

- 信任
- 脆弱度
- 嫉妒
- 希望
- 克制
- 边界

## 常见误区

- 把潜台词写成读者根本看不懂的空白
- 所有人一到感情戏就只会说漂亮话
- 只有吸引，没有防御和风险
- 只靠肢体接触制造张力，忽略语言和选择
