# 都市娱乐文写作指南

## 一、金手指模糊化

### 1. 定义
将主角的先知性（如重生）进行模糊化处理，避免后期束缚

### 2. 方法
采用梦的方式，在关键时刻来完成剧情的自洽与闭环

### 3. 优势
- 避免前期无止境消耗重生者的先知先觉
- 后期不会让读者觉得重复套路
- 作者可以灵活控制金手指的使用时机

### 4. 注意事项
需要完整的前期剧情架构，找准时间线和主角刚重生时发生的事情

## 二、配角情景剧理论

### 1. 核心观点
在主线与主线之间填充有趣的日常剧情，丰富配角性格

### 2. 方法
- 将日常设定在某一个空间里，写好、写尽、写透
- 积累有趣的素材，如短视频、网络梗等
- 通过日常剧情深化配角性格，如"怨种"属性

### 3. 示例
主角拉着配角们去吃饭，其中一个配角痛风坐轮椅，大家互相吐槽、开玩笑的场景

### 4. 好处
- 满足更新量
- 增加读者粘性
- 让读者感受到生活气息

## 三、角色塑造

### 1. 核心
反差与共情

### 2. 方法
- 赋予角色普通人的特质，避免读者产生距离感
- 通过日常细节展现角色的另一面
- 创造窥探欲，让读者了解角色私底下的模样

### 3. 示例
众王之王微服私访，在小摊茶肆吃一碗面，接受旁人递来的蒜，与普通人聊天

### 4. 本质
都市娱乐文的本质是帮读者放大不切实际的想法，在书里完成一场美梦
