# 电竞文写作指南

## 一、舞台的搭建技巧

### 1. 舞台的定义
- **舞台**：供主角表演的地方，并且台下有无数的观众
- **爽感来源**：舞台的大小和高度

### 2. 舞台的大小
- **定义**：观众的数量
- **方法**：
  - 增加观众数量，如直播间观众、现场观众、全球观众等
  - 增加相关人物，如室友、朋友、家人等

### 3. 舞台的高度
- **定义**：对手的强度和事件的重要程度
- **方法**：
  - 增加对手的强度，如从普通玩家到知名主播到世界冠军
  - 增加事件的重要程度，如从晋级赛到联赛到世界赛
  - 增加意义，如背负赛区希望、家人期望等

### 4. 示例应用
- **普通大师晋级赛** → 扩大舞台：增加白银室友，让他们见证主角的崛起
- **大师晋级赛** → 抬高舞台：排到知名主播，被直播间观众记录，视频爆火
- **S赛决赛** → 抬高舞台：铺垫SKT的强大，其他中国队伍被淘汰，队友托孤，父母到场观看

### 5. 核心原则
- 在主角能装逼的时候，在不崩的前提下，观众能拉多少就拉多少，意义能附加多少就附加多少
- 避免写平平无奇的白开水剧情，要给舞台做大、抬高
