# 创作特征提取工具

## 工具简介

本工具用于从用户提供的小说文本中全面提取创作特征，涵盖从叙事节奏到人物塑造的所有细节习惯，帮助作者保持写作风格的一致性，提升作品的独特性和辨识度。

## 提取原则

- **精而不多**：只提取最具代表性、最独特的特征
- **均衡采样**：从文本不同区域采样（开头300行、中间400行、结尾300行）
- **分类存储**：将不同类型的特征存储在不同文件中，便于查阅

## 提取内容与存储结构

### 1. 叙事节奏与结构特征

**存储位置**：`.novel/style/01_narrative_structure.md`

#### 1.1 章节过渡方式
- **章节开头模式**：如何开始每一章（场景描写、对话、内心独白等）
- **章节结尾模式**：如何结束每一章（悬念、总结、留白等）

- **时间跨度处理**：如何处理时间的跳跃和流逝

#### 1.2 段落与句子节奏
- **段落长度偏好**：平均段落长度、长段落与短段落的使用场景
- **句子长度分布**：长句、短句的使用比例和场景
- **标点符号偏好**：逗号、句号、感叹号、问号、省略号的使用习惯
- **节奏变化模式**：紧张场景与舒缓场景的节奏切换方式

## 使用方法

### 1. 提取流程

1. **文本采样**：
   - 开头取300行
   - 中间取400行（选择故事发展的关键部分）
   - 结尾取300行

2. **特征提取**：
   - 按照7个维度逐一分析
   - 每个维度只提取最独特、最具代表性的3-5个特征
   - 每个特征都配1-2个原文示例

3. **文件更新**：
   - 将提取结果分别存储到7个对应文件中
   - 每个文件都有清晰的分类和示例

### 2. 应用建议

1. **写作前参考**：在开始新章节写作前，查阅相关的特征文件
2. **风格一致性检查**：定期对照特征文件检查新写的内容
3. **特征进化**：随着写作进行，定期更新特征文件
4. **针对性优化**：根据提取结果，有针对性地强化个人特色

## 工具价值

1. **风格深度分析**：不仅仅是表面的语料，而是深入到创作习惯的核心
2. **实用性强**：分类清晰，便于快速查阅和应用
3. **精简易读**：每个文件只包含最核心的特征，避免信息过载
4. **全面覆盖**：涵盖创作的各个方面，从结构到细节
