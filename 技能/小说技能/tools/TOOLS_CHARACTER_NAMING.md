# 人物取名工具

## 1. 工具功能

本工具用于为小说中的人物生成符合时代背景和文化特色的名字，确保名字的合理性和真实性。

## 2. 使用方法

### 2.1 核心步骤

1. **确定命名年份**：使用当前年份减1作为搜索年份（例如2026年则搜索2025年取名推荐）
2. **联网搜索**：搜索"[年份]推荐取名"，获取最新的取名推荐列表
3. **筛选分类**：
   - 中式名字：注重姓氏与名字的搭配，考虑五行、寓意等因素
   - 西式名字：注重名字的发音、含义和流行度
4. **随机选择**：从筛选后的列表中随机选择一个名字
5. **验证完整性**：确保名字包含姓和名（对于需要姓氏的角色）

### 2.2 具体操作流程

1. **确定人物背景**：明确人物的国籍、文化背景、时代背景等
2. **选择命名类型**：根据人物背景选择中式或西式名字
3. **执行搜索**：使用相应的关键词进行联网搜索
4. **整理结果**：将搜索结果整理成名字列表，标注每个名字的含义和适用场景
5. **随机选择**：使用随机数生成器从列表中选择一个名字
6. **记录使用**：将使用的名字记录到人物档案中，避免重复使用

## 4. 注意事项

1. **时代背景**：确保名字符合人物所处的时代背景，避免出现时代错误
2. **文化差异**：中式名字和西式名字在结构和含义上有很大差异，应根据人物背景选择合适的命名方式
3. **姓氏完整性**：对于需要完整姓名的角色，确保搜索结果包含姓氏，或根据需要添加合适的姓氏
4. **名字含义**：了解名字的含义，确保其与人物性格、命运等相匹配
5. **避免重复**：在同一部小说中，避免使用相同或相似的名字
6. **搜索结果管理**：将搜索结果存储到知识库中，便于后续使用和参考

## 5. 存储结构

将搜索结果存储在以下目录结构中：

```
.novel/knowledge_base/naming/
├── chinese_names.md      # 中式名字列表
├── western_names.md      # 西式名字列表
└── search_history.md     # 搜索历史记录
```

## 7. 质量控制

1. **多源验证**：从多个来源获取名字推荐，确保名字的流行度和合理性
2. **含义检查**：确保名字的含义积极正面，符合人物形象
3. **时代验证**：确保名字符合人物所处的时代背景
4. **文化验证**：确保名字符合人物的文化背景

## 8. 工具使用频率

- 每次创建新人物时使用本工具
- 定期更新名字数据库，确保使用最新的取名趋势
- 在创作过程中，如发现名字不合适，可重新使用本工具生成新名字
