# 原著资料提取工具

## 功能概述

本工具用于从各类原著资料（如Wiki、官方文档、游戏资料等）中提取完整的世界观和设定集，为同人小说创作提供丰富的参考资料。

## 适用场景

- 创作同人小说时需要提取原著的世界观、设定、人物、地点等信息
- 构建完整的原著资料库，为创作提供参考
- 确保同人创作与原著设定的一致性

## 工作流程

### 1. 资料来源选择

选择权威的原著资料来源，如：
- 官方Wiki（中文和英文）
- 官方游戏资料
- 官方小说和漫画
- 官方动画和电影

### 2. 资料提取步骤

1. **制定提取计划**：
   - 确定需要提取的核心内容（世界观、人物、地点、时间线等）
   - 规划需要读取的页面和资料

2. **资料读取与整理**：
   - 系统性地读取相关Wiki页面
   - 记录所有读取的页面，确保覆盖所有重要内容
   - 对读取的内容进行分类整理

3. **内容翻译与整合**：
   - 将英文内容翻译为中文
   - 优先使用中文资料，英文资料作为补充
   - 整合不同来源的信息，确保一致性

4. **资料存储与管理**：
   - 按照项目管理系统的结构存储提取的资料
   - 创建结构化的Markdown文档
   - 建立索引和目录，方便查阅

## 提取内容结构

### 1. 世界观设定
- 核心概念
- 历史背景
- 社会结构
- 经济体系
- 文化特色
- 地理环境

### 2. 重要人物
- 主要角色
- 次要角色
- 人物关系
- 人物背景和经历

### 3. 组织与势力
- 巨型企业
- 帮派和组织
- 执法机构
- 其他重要团体

### 4. 时间线
- 重要历史事件
- 时间顺序整理
- 关键节点标记

### 5. 重要地点
- 主要城市和区域
- 重要建筑和场所
- 地理特征

### 6. 技术与装备
- 赛博改造
- 武器系统
- 载具
- 网络技术
- 其他科技

## 工具使用注意事项

1. **中文优先**：优先使用中文资料，英文资料作为补充
2. **资料完整性**：确保覆盖所有重要的原著内容
3. **准确性**：确保提取的信息与原著一致
4. **结构化**：按照标准结构整理资料，便于查阅和使用
5. **更新维护**：定期更新资料，确保与最新原著内容同步

## 示例：赛博朋克2077和边缘行者资料提取

### 资料来源
- 中文Wiki：https://cyberpunk.huijiwiki.com
- 英文Wiki：https://cyberpunk.fandom.com

### 提取内容
- 世界观设定（赛博朋克世界的核心概念、历史、社会结构等）
- 重要人物（游戏和动画中的主要角色）
- 组织与势力（荒坂、军用科技等）
- 时间线（从1919年到2077年的重要事件）
- 重要地点（夜之城及其各区）
- 技术与装备（赛博改造、武器、载具等）

### 存储结构
```
.novel/
└── source_material/
    ├── worldview.md         # 世界观设定
    ├── characters.md        # 重要人物
    ├── organizations.md     # 组织与势力
    ├── timeline.md          # 时间线
    ├── locations.md         # 重要地点
    ├── technology.md        # 技术与装备
    ├── page_read_log.md     # 页面读取日志
    └── index.md             # 总索引
```

## 总结

原著资料提取工具是同人小说创作的重要辅助工具，通过系统性地提取和整理原著资料，可以为创作提供丰富的参考，确保同人作品与原著设定的一致性，同时也为创作者节省大量的资料收集时间。
